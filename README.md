# 🇷🇺 Rime-Jcuken (Русский ЙЦУКЕН)

> 纯正的俄罗斯传统物理键盘体验，专为 Rime (中州韵/小狼毫/鼠须管) 打造。
> Аутентичная русская раскладка ЙЦУКЕН, созданная специально для Rime.

---

## ✨ 特性 (Features / Особенности)

这是为习惯传统俄语物理键盘的 native speaker 独家定制的专属方案！不再需要痛苦地适应音译键盘，闭着眼睛也能敲出纯正的俄语。
Это эксклюзивное решение для носителей языка, привыкших к традиционной русской физической клавиатуре! Вам больше не нужно мучиться с транслитом — печатайте на чистом русском даже с закрытыми глазами.

- 🎹 **1:1 原生键盘映射 (Нативная раскладка 1:1)**: 完美复刻标准 ЙЦУКЕН 键盘。按 `Q` 出 `й`，按 `F` 出 `а`。标点符号也完全遵循原生布局。 (Идеальная копия стандартной клавиатуры ЙЦУКЕН. Знаки препинания также полностью соответствуют оригинальной раскладке.)
- 🚀 **万词级智能联想 (Умный ввод на 10 000 слов)**: 内置俄罗斯最常用的 10,000 高频词汇。你只需敲打单词的前缀，大长单词便会自动浮现。(Встроено 10 000 самых частотных русских слов. Просто наберите начало слова, и длинные слова появятся автоматически.)
- 🖨️ **无缝打字机兜底 (Бесшовный режим пишущей машинки)**: 遇到词库里没有的生僻词、名字或俚语？底层的 Lua 脚本会自动为你将按键**强行翻译**为纯正的俄文字母，绝不卡壳。(Встретили редкое слово или сленг, которого нет в словаре? Встроенный Lua-скрипт автоматически переведет нажатия клавиш в русские буквы без задержек.)
- 🎨 **专属 Emoji 彩蛋 (Эксклюзивные Emoji)**: 附带定制的“俄语-Emoji”联想功能。输入 `любовь` 出 ❤️，输入 `россия` 出 🇷🇺。(Набирайте `любовь` — получайте ❤️, набирайте `россия` — получайте 🇷🇺.)

## 📦 安装指南 (Установка)

1. **下载本项目的所有文件**。(Скачайте все файлы этого проекта.)
2. 将以下文件放入你的 Rime 用户文件夹 (Поместите следующие файлы в папку пользователя Rime):
   - `jcuken.schema.yaml`
   - `jcuken.dict.yaml`
   - `ru_literal.lua`
3. 将 `opencc` 文件夹中的文件放入 Rime 的 `opencc` 目录中 (Поместите файлы из папки `opencc` в директорию `opencc` вашего Rime):
   - `opencc/ru_emoji.txt`
   - `opencc/ru_emoji.json`
4. 在你的 `rime.lua` 文件中添加以下代码以激活底层兜底脚本 (Добавьте следующий код в ваш файл `rime.lua` для активации скрипта):
   ```lua
   ru_literal = require("ru_literal")
   ```
5. 在你的 `default.custom.yaml` 中添加本方案 (Добавьте эту схему в ваш `default.custom.yaml`):
   ```yaml
   patch:
     schema_list:
       - schema: jcuken
       # ... 你原有的其他方案 (другие ваши схемы)
   ```
6. **重新部署 (Deploy / Перезапуск)**，选择 **Русский (ЙЦУКЕН)** 即可开始盲打！(Выберите **Русский (ЙЦУКЕН)** и начните печатать вслепую!)
